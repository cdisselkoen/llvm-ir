impl Typed for Terminator[src]

The Type of a Terminator is its result type. For most terminators, this is VoidType. For instance, a Ret instruction has void type even if the function returns a non-void value; we do not store the result of a Ret instruction using something like %3 = ret i32 %2. See LLVM 9 docs on Terminator Instructions
